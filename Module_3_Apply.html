The reading gives two reasons why: 1. **It matches the problem type.** Regression means predicting a number (like a house price), and MSE is designed specifically for that — as opposed to cross-entropy, which is for classification problems where you're predicting a category. 2. **It penalizes big mistakes heavily.** Because MSE *squares* each error before averaging them, a few enormous misses hurt the loss far more than many small ones. The reading explains this as a feature, not a bug — it "pushes the model to avoid big mistakes."
