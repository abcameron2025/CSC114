# CSC114 – Artificial Intelligence I

Welcome to the repository for **CSC114: Artificial Intelligence I**. This course introduces the foundational concepts of Artificial Intelligence, intelligent agent design, machine learning workflows, and modern AI development tools.

## 📘 Course Description

CSC114 explores the study of intelligent agent design and rational decision-making systems. Topics include:

- Goal-driven agents
- Search techniques and optimization
- Problem-solving methods
- Logic and knowledge-based systems
- Statistical and probabilistic reasoning
- Fundamentals of Machine Learning and Deep Learning
- Computer Vision
- Natural Language Processing (NLP)
- Large Language Models (LLMs)

Upon completion of the course, students will be able to demonstrate core AI design concepts and apply practical AI development skills.

---

## 🧠 Learning Outcomes

Students in this course will:

- Create custom AI agents
- Understand the Machine Learning workflow
- Apply classification and regression algorithms
- Learn and use AI frameworks such as Keras and PyTorch
- Explore Deep Learning fundamentals
- Develop introductory Computer Vision applications
- Apply Natural Language Processing techniques
- Learn how Large Language Models operate and can be optimized
